MoneyManagerEx for Android

Introduction

Money Manager Ex for Android is a port of the desktop personal finance application Money Manager Ex, currently available for Windows, Mac OSX and Linux. The Android versiown mirrors the best features of the desktop version, while adding mobility and the database synchronization.

Documentation

See our Wiki pages for documentation and tutorials.

A User Manual is a useful guide for newcomers and experienced users alike.

Download

You can download the app from our Releases page.

The app is available through Google Play Store and Amazon App Store.

Implemented features

  • Transaction management
  • Multiple accounts and account types, currencies, and languages.
  • Account, payee, category, currency management
  • Budgets (view only, currently)
  • Investment watchlists
  • Storage card Import/Export
  • Dropbox synchronization
  • Security - application lock/unlock with pin code
  • Recurring transactions
  • Reports and Charts

Planned features

  • Database encryption support (prototype ready!)
  • Stocks
  • Attachments
  • and more... Click to see our list and/or submit a feature request or a bug report.

Pre-Release Versions / Testing

Should you want to try out the latest version of Money Manager Ex for Android, you can access it in two ways:

  • From our Releases page on GitHub. Simply download and install the latest version manually. You need to enable installing from unknown sources in Android settings first.
  • Opt-in at Google Play. This way you will receive the latest updates through the Play Store. You can leave the Beta Testing program at the same link.
